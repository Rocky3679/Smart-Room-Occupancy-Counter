# Smart Room Occupancy Counter

An Arduino-based intelligent room occupancy monitoring system that automatically counts people entering and leaving a room while automatically controlling electrical appliances based on occupancy.

---

# Project Overview

This project was developed to demonstrate how embedded systems can be used for intelligent energy management in buildings.

Using two infrared sensors, an Arduino UNO continuously monitors the movement of people entering and exiting a room. The occupancy count is displayed on an OLED display. When at least one person is present, the lighting system is turned ON automatically through a relay module. When the room becomes empty, the light turns OFF automatically.
